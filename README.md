# Retail Sales Analysis Dashboard

## Project Overview
This project analyzes retail sales performance, customer behavior, inventory metrics, and product trends using Power BI.

## Tools Used
- Power BI
- DAX
- Power Query
- Excel

## Dashboard Modules

### 1. Executive Dashboard
- Provides an overall summary of retail business performance
- Tracks sales, profit, revenue, and key KPIs
- Helps monitor overall operational performance

### 2. Customer Insights Page
- Analyzes customer purchasing behavior and trends
- Identifies high-value customers and sales patterns
- Supports customer-focused business decisions

### 3. Inventory Dashboard
- Monitors inventory levels and stock availability
- Identifies low-stock and high-demand products
- Helps improve inventory management

### 4. Product Analysis
- Analyzes product-wise sales and profitability
- Identifies top-performing and low-performing products
- Tracks category and product trends

## Key Insights
- Identified top-selling products and profitable categories
- Analyzed customer purchasing behavior
- Monitored inventory performance and stock trends
- Evaluated overall sales and revenue growth
